append(fields, replace) → {this}

Append one or more fields to the RequestBody. If a fields is a string, this method acts as an alias of module:maltypart.RequestBody#setField.

Parameters:
Name Type Argument Default Description
fields Array | Object

An Array of objects with name and value properties, or a key-value Object.
replace Boolean <optional>
 true

By default, any existing field with the same name is replaced. If replace is set to false, existing values will be left in-place, and the new value will be appended. This results in duplicate keys (which can be useful!).
